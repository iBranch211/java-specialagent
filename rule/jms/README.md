# SpecialAgent Rule for JMS

**Rule Name:** `jms`

## Compatibility

### JMS v1

```xml
<groupId>javax.jms</groupId>
<artifactId>jms-api</artifactId>
<version>[1.1-rev-1,LATEST]</version>
```

### JMS v2

```xml
<groupId>javax.jms</groupId>
<artifactId>javax.jms-api</artifactId>
<version>[2.0.1,LATEST]</version>
```
